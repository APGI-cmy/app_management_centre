# Lessons Learned

## Session 20260408

### Lesson: Do not commit placeholder lesson templates
- Context: When updating agent governance notes or personal lessons-learned records.
- Pattern: Placeholder values such as `YYYYMMDD`, `[Title]`, or bracketed prompts remain in committed Markdown files.
- Action: Replace template markers with concrete session details before committing, or leave the file out of the PR until real learnings are available.
