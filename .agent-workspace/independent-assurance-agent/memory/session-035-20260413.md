   issues that IAA did not detect (or that were not visible at the time of the prior session).
   This is correct governance: CS2 is the final authority. IAA re-invocation after CS2 correction
   directives is the correct protocol, not a sign of IAA failure.

   uncorrected, CodexAdvisor would route consumer-repo propagation to the wrong repository.
   IAA should treat agent name correctness in `can_invoke` blocks as a substantive check,
   not merely a format check.

   escalation mechanism in SELF-MOD-001 is a constitutional requirement. A compressed version
